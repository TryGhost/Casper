My process for updating the fonts:
- Go to https://icomoon.io/app/
- Import casper-icons.svg
- Select all imported characters for export, plus new character(s)
- Click on "Generate Font" (near the bottom)
- Change character codes/names of new icons as needed
  - code(s) for copy and paste in html: f60a="" (reddit)
- Click on Preferences (gear icon)
- Change Font-Name to "icons"
  (despite "icomoon" in the SVG, it was "icons" in the TTF)
- Select "Font Metrics"
- Change "Em Square Height" to 512
- Select "version"
- Bump version to higher number
- Close settings
- Click download
- Unzip results, compare to verify format is correct
- rename and overwrite files
- Bumped version in casper/assets/css/screen.css ("?v=1" to "?v=2")
  See the comment in screen.css (would 1.1 have worked?)
